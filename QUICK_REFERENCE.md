## ğŸ”§ Troubleshooting

### Data Not Saving:
- Check browser's localStorage is enabled
- Check available storage space
- Try exporting and re-importing data

### Drag & Drop Not Working:
- Ensure "Manual Order" is selected in sort options
- Not available for archived todos
- Refresh the page if issues persist

### Time Tracking Issues:
- Only one timer can run per todo at a time
- Make sure to stop timer before closing browser
- Time entries are saved immediately

### Import Failed:
- Ensure file is valid JSON format
- Check file wasn't corrupted during download
- File must have been exported from this app
